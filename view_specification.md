# 图谱视图规范（view_specification）

## 1. 模式定义
- `focus`：焦点树（2 跳）。
- `paternal`：父系分支视图。
- `maternal`：母系分支视图。
- `overview`：全景视图。

## 2. 当前筛选规则
- 关系类型：`parent_child`、`sibling`。
- 顶部可见性开关：父母子女、兄弟姐妹。
- `配偶`开关已移除（夫妻已合并为单元卡）。

## 3. 布局规则
- 全景：允许手工拖拽，位置持久化。
- 非全景：自动布局。
- 同代单元卡：Y 对齐。
- 同代单元卡：禁止重叠（自动避让）。

## 4. 连线规则
- 默认无自动线（迁移不再生成）。
- 用户可自由创建/删除/重连。
- 命中范围增强：交互宽度与连接半径扩大。

## 5. 焦点规则
- 点单元卡：设焦点单元。
- 点头像：选中个人并同步焦点单元。

## 6. 父系/母系规则（当前）
- 从焦点单元向上找父母单元候选。
- 基于成员姓名关键词识别父系/母系 root。
- 沿 `parent_child` 与必要 sibling 扩展可见集合。
